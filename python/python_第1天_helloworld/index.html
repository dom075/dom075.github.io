Python 第1天 HelloWorld

Jun 1, 2020 · 1 min read

Python特点

  1. 解释型语言，对两种主流的编程范式（面向对象和函数式编程）都提供支持。
  2. 执行效率较低，代码无法加密。
  3. 开发时可选择的框架太多。

安装Python解释器

可以在Python官网下载。
安装过程网上很多，查一下就行。
安装好之后确认版本。

python --version

Hello World

可以用文本编辑工具（推荐使用Sublime、Visual Studio Code等高级文本编辑工具）编写Python源代码并用py作为后缀名保存该文件，代码内容如下所示。

print('hello, world!')

然后切换到源代码所在的目录并执行下面的命令，看看屏幕上是否输出了"hello, world!"。
* 注意这个文件不能被别的程序打开着，不然会提示找不到这个文件。
