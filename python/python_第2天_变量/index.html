Jun 2, 2020 · 1 min read

Python常用变量

  • 整型： Python 3.x之后只有int了，可以处理任意大小的整数，同时也支持二进制、八进制、十六进制等等。
  • 浮点型： 小数（支持一般的122.45和科学计数法 1.2e2）.
  • 字符串： ‘hello’ “hello” 多行要用三个单引号或双引号。
  • 布尔型： True / False 或 2 == 1 这种运算
