Python 第11天 读写文件和Json

Jun 11, 2020 · 2 min read

open方法

操作模式 具体含义
'r' 读取 （默认）
'w' 写入（会先截断之前的内容）
'x' 写入，如果文件已经存在会产生异常
'a' 追加，将内容写入到已有文件的末尾
'b' 二进制模式
't' 文本模式（默认）
'+' 更新（既可以读又可以写）

读取文件内容

def main():
    f = None
    try:
        f = open('C:\Blog\Python\Python_第3天_If.md', 'r', encoding='utf-8')
        print(f.read())
    except FileNotFoundError:
        print('无法打开指定的文件!')
    except LookupError:
        print('指定了未知的编码!')
    except UnicodeDecodeError:
        print('读取文件时解码错误!')
    finally:
        if f:
            f.close()


if __name__ == '__main__':
    main()

try catch 里不想用finally的话可以用with关键字

with的原理：紧跟with后面的语句会被求值，返回对象的__enter__()方法被调用，这个方法的返回值将被赋值给as关键字后面的变量，当with后面的代码块全部被执行完之后，将调用前面返回对象的__exit__()方法。
也就是说，with关键就是使用的对象必须要有__enter__()和__exit__()这两个方法

def main():
    try:
        with open('致橡树.txt', 'r', encoding='utf-8') as f:
            print(f.read())
    except FileNotFoundError:
        print('无法打开指定的文件!')
    except LookupError:
        print('指定了未知的编码!')
    except UnicodeDecodeError:
        print('读取文件时解码错误!')


if __name__ == '__main__':
    main()

读取文件后的对象可以直接使用，也可以放到List中，也可以直接循环每行来使用。

# 一次性读取整个文件内容
with open('致橡树.txt', 'r', encoding='utf-8') as f:
    print(f.read())

# 通过for-in循环逐行读取
with open('致橡树.txt', mode='r') as f:
    for line in f:
        print(line, end='')
        time.sleep(0.5)
print()

# 读取文件按行读取到列表中
with open('致橡树.txt') as f:
    lines = f.readlines()
print(lines)

读写二进制文件（比如图片）

def main():
    try:
        with open('guido.jpg', 'rb') as fs1:
            data = fs1.read()
            print(type(data))  # <class 'bytes'>
        with open('吉多.jpg', 'wb') as fs2:
            fs2.write(data)
    except FileNotFoundError as e:
        print('指定的文件无法打开.')
    except IOError as e:
        print('读写文件时出现错误.')
    print('程序执行结束.')

if __name__ == '__main__':
    main()

读写JSON文件

Json格式和Python格式的对应：

JSON Python
object dict
array list
string str
number (int / real) int / float
true / false True / False
null None

Json格式存储文件

import json
def main():
    mydict = {
        'name': '骆昊',
        'age': 38,
        'qq': 957658,
        'friends': ['王大锤', '白元芳'],
        'cars': [
            {'brand': 'BYD', 'max_speed': 180},
            {'brand': 'Audi', 'max_speed': 280},
            {'brand': 'Benz', 'max_speed': 320}
        ]
    }
    try:
        with open('data.json', 'w', encoding='utf-8') as fs:
            json.dump(mydict, fs)
    except IOError as e:
        print(e)
    print('保存数据完成!')

if __name__ == '__main__':
    main()

json的序列化（serialization）和 反序列化（deserialization）

  • dump - 将Python对象按照JSON格式序列化到文件中
  • dumps - 将Python对象处理成JSON格式的字符串
  • load - 将文件中的JSON数据反序列化成对象
  • loads - 将字符串的内容反序列化成Python对象
