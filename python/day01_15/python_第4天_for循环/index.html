Python 第4天 for循环

Jul 4, 2020 · 1 min read

for-in循环

sum = 0
for x in range(101):
    sum += x
print(sum)

'一次性抽出index和element'
for index, elem in enumerate(list1):
    print(index, elem)

while循环

用break退出，continue跳到下一个循环。

import random

answer = random.randint(1, 100)
counter = 0
while True:
    counter += 1
    number = int(input('请输入: '))
    if number < answer:
        print('大一点')
    elif number > answer:
        print('小一点')
    else:
        print('恭喜你猜对了!')
        break
print('你总共猜了%d次' % counter)
