方法定义

  1. 用def关键字来申明方法。
def fac(num):
    result = 1
    for n in range(1, num + 1):
        result *= n
    return result


m = int(input('m = '))
n = int(input('n = '))

print(fac(m) // fac(n) // fac(m - n))
  1. Python可以重复定义方法（后面的声明覆盖前面的声明），调用时选取最后的声明。
def foo():
    print('hello, world!')

foo()

参数

参数可以带默认值，调用方法而不传入参数值时，按照参数次序使用默认参数值。
因此python不需要重载。

def addtest(a=10, b=20, c=30):
    """三个数相加"""
    return a + b + c

print(addtest())
print(addtest(1))
print(addtest(1, 2))
print(addtest(1, 2, 3))
# 传递参数时可以不按照设定的顺序进行传递
print(add(c=50, a=100, b=200))
'（分别输出 60 51 33 6 350）'

可变参数： * 就是把参数当作一个可变长的Array，所以调用时不必在意需要几个参数。

# 在参数名前面的*表示args是一个可变参数
def add(*args):
    total = 0
    for val in args:
        total += val
    return total

模块

已经知道可以用Import导入模块，如果导入的模块当中有执行代码，那导入时就会被执行。
要避免这个问题，就需要判断当前模块是不是正在执行的模块，如果是，则执行，否则不执行。

def foo():
    pass
def bar():
    pass
# __name__是Python中一个隐含的变量它代表了模块的名字
# 只有被Python解释器直接执行的模块的名字才是__main__
if __name__ == '__main__':
    print('call foo()')
    foo()
    print('call bar()')
    bar()

变量的作用域

  • 全局变量：在执行代码中定义的变量就是全局变量，可以在下面调用的方法里直接使用。
  • 局部变量：在方法中的变量就是局部变量。局部变量可以在内嵌方法中被访问。
def foo():
    b = 'hello'

    # Python中可以在函数内部再定义函数
    def bar():
        c = True
        print(a)
        print(b)
        print(c)

    bar()
    # print(c)  # NameError: name 'c' is not defined

if __name__ == '__main__':
    a = 100
    # print(b)  # NameError: name 'b' is not defined
    foo()

通过函数来修改全局变量： 错误方法：(实际上是新定义了一个局部变量a)

def foo():
    a = 200
    print(a)  # 200

if __name__ == '__main__':
    a = 100
    foo()
    print(a)  # 100

正确方法：要加上global才可以(内嵌方法要更改外部方法的变量值，要加上nonlocal关键字)

def foo():
    global a
    a = 200
    print(a)  # 200

if __name__ == '__main__':
    a = 100
    foo()
    print(a)  # 200

提示：开发中尽量少用全局变量。

闭包

Python 中通俗一点来说，如果在一个函数内部，嵌套了函数，这个内部函数对（非全局作用域）外部作用域的变量进行引用，那么这个内部函数称为闭包。闭包每次运行是能记住引用的外部作用域的变量的值。

#闭包函数的实例
# outer是外部函数 a和b都是外函数的临时变量

def outer( a ):
     b = 10
     # inner是内函数
     def inner():
         #在内函数中 用到了外函数的临时变量
         print(a+b)
     # 外函数的返回值是内函数的引用
     return inner
 
if __name__ == '__main__':
     # 在这里我们调用外函数传入参数5
     #此时外函数两个临时变量 a是5 b是10 ，并创建了内函数，然后把内函数的引用返回存给了demo
     # 外函数结束的时候发现内部函数将会用到自己的临时变量，这两个临时变量就不会释放，会绑定给这个内部函数
     demo = outer(5)
     # 我们调用内部函数，看一看内部函数是不是能使用外部函数的临时变量
     # demo存了外函数的返回值，也就是inner函数的引用，这里相当于执行inner函数
     demo() # 15
 
     demo2 = outer(7)
     demo2()#17
